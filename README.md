# business-performance-analytics
Business Performance Analytics

An interactive Power BI dashboard project focused on uncovering patterns in sales performance, profitability, customer segments, and discount behavior using retail business data.

# Project Overview

This project explores how different business factors influence overall company performance through data visualization and KPI tracking. The dashboard was built to simulate real-world business reporting workflows commonly used by analysts and operations teams.

The analysis focuses on:

Sales performance trends over time
Profitability across product categories
Regional and customer segment insights
The relationship between discounts and profit
Executive-style KPI monitoring

# Key Insights
Certain product categories generated significantly higher profit margins than others.
Large discounts did not always translate into higher profitability.
Sales performance fluctuated throughout the year, revealing seasonal business trends.
Customer segments displayed different purchasing and profitability behaviors.
Business Value

This dashboard demonstrates how business intelligence tools can be used to monitor company performance, identify operational trends, and support data-driven decision-making through interactive analytics and KPI reporting.
